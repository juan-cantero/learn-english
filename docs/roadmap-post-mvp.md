# Post-MVP Roadmap

## Phase A: Polish & UX
- [ ] User profile page (edit display name, avatar, preferred difficulty)
- [ ] Toast notifications for actions (classroom created, joined, assignment completed, etc.)
- [ ] Onboarding flow for new users (first-time empty states with guided actions)
- [ ] Loading skeletons across all pages (consistent pattern)
- [ ] Error boundaries with retry actions
- [ ] Keyboard shortcuts for lesson navigation

## Phase B: Core Learning
- [ ] Link assignments to lesson page (Start button opens episode lesson directly)
- [ ] Auto-complete assignment when all exercises in a lesson are finished
- [ ] Score calculation from exercise results feeds back to assignment submission
- [ ] Progress tracking per episode (vocabulary mastered, exercises completed)
- [ ] Spaced repetition for vocabulary review
- [ ] Bookmarking expressions/vocabulary for personal study list

## Phase C: Discovery
- [ ] Search shows by title
- [ ] Filter shows by genre, difficulty level, accent type
- [ ] "Continue Learning" recommendations based on progress history
- [ ] Show difficulty badge and episode count on browse cards
- [ ] Recently studied episodes section on homepage
